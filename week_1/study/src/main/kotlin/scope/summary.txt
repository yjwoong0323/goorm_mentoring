scope func =
  람다를 사용해 일시적인 영역을 만들고 코드를 더 간결하게 만들거나, method chaning에
  활용하는 함수를 의미함

- 확장 함수의 특징은 마치 멤버 함수를 쓰는 것처럼 사용
- let, run : 람다의 결과 반환
- also, apply : 객체 그 자체 반환
   it    this   사용

this : 생략 가능, 다른 이름 붙일 수 없음
it : 생략 불가, 다른 이름 붙일 수 있음

with : 확장 함수 아님. this 사용

<let> : 하나 이상의 함수를 call chain 결과로 호출할 때


----
사용 빈도가 적은 관용구는 코드를 더 복잡하게 만들고 이런 관용구들은 한 문장 내에서 조합해 사용하면
복잡성이 훨씬 증가 함.
하지만 scope func을 사용하면 안된다는 것은 아님, 적절한 convention을 적용하면 유용하게 사용 가능
